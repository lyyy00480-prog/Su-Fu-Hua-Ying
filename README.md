## 苏府画影 Demo 运行说明

### 1. 下载与解压

请从提供的链接下载 `EmotionGameDemo.zip` 文件，并将其解压到您希望存放项目的任意位置。解压后，您会得到一个名为 `EmotionGameDemo` 的文件夹。

### 2. 文件结构

解压后的文件夹内应包含以下文件和目录：

-   `EmotionGameDemo.exe`: 可执行程序。
-   `assets/`: 包含游戏所需的所有图像、字体和音频资源。
-   `data/`: 包含游戏的配置文件和剧本文件。
-   其他 PyInstaller 生成的支持文件。

**重要提示：** 请确保 `EmotionGameDemo.exe` 文件与 `assets/` 和 `data/` 文件夹位于同一层级目录，否则程序可能无法找到资源。

### 3. 运行 Demo

1.  进入解压后的 `EmotionGameDemo` 文件夹。
2.  双击运行 `EmotionGameDemo.exe` 文件。

游戏将以全屏模式启动，并显示菜单界面。您可以通过以下方式与 Demo 互动：

-   **菜单界面**：按 `空格键` 进入游戏。
-   **游戏界面**：
    -   点击鼠标左键或按 `空格键` 切换到下一句台词。
    -   按 `F11` 键可以在全屏和窗口模式之间切换。

### 4. 常见问题与解决方案

-   **程序无法启动或报错**：
    -   请检查 `EmotionGameDemo.exe` 是否与 `assets/` 和 `data/` 文件夹位于同一目录。
    -   尝试更新您的显卡驱动。
-   **背景音乐或音效不播放**：
    -   请检查您的系统音量设置。
    -   确保 `assets/audio/` 文件夹中包含有效的音频文件。
