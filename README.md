# Emotion Game Project: Su Mansion's Painting Shadow (苏府画影)

This repository contains the core development of the visual novel project "Su Mansion's Painting Shadow" (苏府画影).

## Project Structure (项目结构)

The project is organized into the following main directories, each with specific responsibilities:
本项目分为以下几个主要目录，每个目录承担着特定的职责：

-   `core/`: **Game Core Engine Logic (游戏核心引擎逻辑)**
    -   Contains fundamental game mechanics, such as asset management, dialogue system, and state management.
    -   **Important:** Non-technical personnel should **NOT** modify files in this directory.
    -   包含游戏的基础机制，如资源管理、对话系统和状态管理。
    -   **重要提示：** 非技术人员请勿修改此目录中的文件。

-   `assets/`: **Art and Audio Assets Library (美术与音频资产库)**
    -   Stores all visual and auditory assets, including character sprites, background images, background music (BGM), sound effects, and fonts.
    -   美术和音频团队在此存放所有视觉和听觉素材，包括角色立绘、背景图、背景音乐（BGM）、音效和字体等。

-   `data/`: **Script and Configuration Files (剧本与配置文件)**
    -   Houses JSON-formatted game scripts (`script.json`) and asset configuration (`assets_config.json`).
    -   **Important:** Planners and writers should edit story content and asset mappings here.
    -   存放 JSON 格式的游戏剧本文件（`script.json`）和资源配置文件（`assets_config.json`）。
    -   **重要提示：** 策划和编剧人员在此编辑剧情内容和素材映射。

-   `scripts/`: **Development Helper Tools (开发辅助工具)**
    -   Contains utility scripts for development, such as asset validation tools (`validator.py`) and test asset generation (`generate_test_assets.py`).
    -   包含开发过程中使用的辅助脚本，例如素材校验工具（`validator.py`）和测试素材生成脚本（`generate_test_assets.py`）。

## Demo Usage (Demo 使用说明)

### 1. Download and Unzip (下载与解压)

Download the `EmotionGameDemo.zip` file and extract it to any desired location. After extraction, you will find a folder named `EmotionGameDemo`.

请从提供的链接下载 `EmotionGameDemo.zip` 文件，并将其解压到您希望存放项目的任意位置。解压后，您会得到一个名为 `EmotionGameDemo` 的文件夹。

### 2. File Structure of Demo (Demo 文件结构)

The extracted folder should contain the following files and directories:
解压后的文件夹内应包含以下文件和目录：

-   `EmotionGameDemo.exe`: The executable application.
-   `assets/`: Contains all necessary image, font, and audio resources for the game.
-   `data/`: Contains game configuration and script files.
-   Other supporting files generated by PyInstaller.

**Important:** Ensure that `EmotionGameDemo.exe` is in the same directory level as the `assets/` and `data/` folders, otherwise the program may not be able to locate resources.

**重要提示：** 请确保 `EmotionGameDemo.exe` 文件与 `assets/` 和 `data/` 文件夹位于同一层级目录，否则程序可能无法找到资源。

### 3. Run the Demo (运行 Demo)

1.  Navigate to the extracted `EmotionGameDemo` folder.
2.  Double-click to run `EmotionGameDemo.exe`.

The game will launch in fullscreen mode, displaying the menu screen. You can interact with the Demo as follows:

1.  进入解压后的 `EmotionGameDemo` 文件夹。
2.  双击运行 `EmotionGameDemo.exe` 文件。

游戏将以全屏模式启动，并显示菜单界面。您可以通过以下方式与 Demo 互动：

-   **Menu Screen (菜单界面)**: Press `Space` to start the game.
    -   按 `空格键` 进入游戏。
-   **Game Screen (游戏界面)**:
    -   Click the left mouse button or press `Space` to advance to the next line of dialogue.
    -   Press `F11` to toggle between fullscreen and windowed mode.
    -   点击鼠标左键或按 `空格键` 切换到下一句台词。
    -   按 `F11` 键可以在全屏和窗口模式之间切换。

### 4. Common Issues and Solutions (常见问题与解决方案)

-   **Program fails to start or reports errors (程序无法启动或报错)**:
    -   Verify that `EmotionGameDemo.exe` is in the same directory as the `assets/` and `data/` folders.
    -   Try updating your graphics card drivers.
    -   请检查 `EmotionGameDemo.exe` 是否与 `assets/` 和 `data/` 文件夹位于同一目录。
    -   尝试更新您的显卡驱动。
-   **Background music or sound effects are not playing (背景音乐或音效不播放)**:
    -   Check your system's volume settings.
    -   Ensure that valid audio files are present in the `assets/audio/` folder.
    -   请检查您的系统音量设置。
    -   确保 `assets/audio/` 文件夹中包含有效的音频文件。

If you encounter any issues during operation, please feel free to contact me.

如果您在运行过程中遇到任何问题，请随时联系我。
